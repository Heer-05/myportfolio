## 🎮 INTERACTIVE SYSTEMS

### Parallax System
- **Trigger**: Mouse movement
- **Effect**: Radial gradient follows cursor
- **Calculation**: (mouseX - centerX) * 0.0005
- **Performance**: 60fps smooth
- **Responsiveness**: Real-time tracking

### Scroll System
- **Trigger**: Window scroll
- **Effect**: Atmospheric parallax + section enhancement
- **Performance**: Throttled, optimized
- **Responsiveness**: Smooth depth shifts

### Performance Monitor
- **Monitoring**: Real-time FPS detection
- **Threshold**: 50fps
- **Action**: Auto-reduces intensity if FPS drops
- **Recovery**: Auto-restores when FPS recovers

### Accessibility System
- **Detection**: prefers-reduced-motion media query
- **Action**: Disables all animations
- **Fallback**: Static visual structure maintained
- **Compliance**: WCAG 2.1 Level AA
