## ANIMATION REFERENCE

### Testimonials
- **Card Entrance**: Fade-in-up (0.8s ease-out)
- **Card Hover**: Lift (-4px) + shadow (0.3s)
- **Carousel Transition**: Fade (0.3s)
- **Dot Hover**: Subtle glow (0.3s)
- **Dot Active**: Champagne Silk with box-shadow

### Contact
- **Section Entrance**: Fade-in-up (0.8s ease-out)
- **Link Hover**: Lift (-4px) + shadow (0.3s)
- **Input Focus**: Border glow (0.3s)
- **Button Hover**: Lift (-2px) + shadow (0.3s)

### Footer
- **Subtle**: Radial gradient overlay (no animation)

## RESPONSIVE BREAKPOINTS

### Desktop (769px+)
- Testimonials: 3-column grid
- Contact: 2-column layout
- Full spacing and typography
- All elements visible

### Tablet (768px)
- Testimonials: Single column with carousel
- Contact: Single column stacked
- Reduced padding (2xl → lg)
- Reduced font sizes (2.8rem → 2rem)

### Mobile (480px)
- Testimonials: Single card carousel
- Contact: Single column stacked
- Minimal padding (lg)
- Optimized font sizes (1.6rem headings)
- Touch-friendly buttons (48px)

## INTERACTION PATTERNS

### Testimonials Carousel
1. **Desktop**: All 3 cards visible in grid
2. **Mobile**: Single card displayed
3. **Navigation**: Click dots to change card
4. **Keyboard**: Arrow keys to navigate
5. **Transition**: Smooth fade (0.3s)

### Contact Form
1. **Focus**: Input glows with Champagne Silk
2. **Hover**: Button lifts and glows
3. **Submit**: Form resets with feedback
4. **Validation**: Required fields enforced

### Contact Links
1. **Hover**: Card lifts (-4px)
2. **Shadow**: Enhanced on hover
3. **Border**: Champagne Silk on hover
4. **Click**: Opens email/social link
