## 📝 TYPOGRAPHY SYSTEM

### Font Stack

#### Heading Font: Cormorant Garamond
- **Family**: Serif, Editorial
- **Weight**: 400 (Regular), 500 (Medium), 600 (Semibold)
- **Usage**: H1, H2, H3, Display text
- **Characteristics**: Large, elegant, luxury fashion-inspired
- **Import**: `@import url('https://fonts.googleapis.com/css2?family=Cormorant+Garamond:wght@400;500;600&display=swap');`

#### Sub-heading Font: Montserrat
- **Family**: Sans-serif, Geometric
- **Weight**: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
- **Usage**: H4, H5, Labels, Tags, Navigation
- **Characteristics**: Clean, geometric, modern luxury
- **Import**: `@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&display=swap');`

#### Body Font: Inter
- **Family**: Sans-serif, Humanist
- **Weight**: 400 (Regular), 500 (Medium)
- **Usage**: Body text, descriptions, paragraphs
- **Characteristics**: Highly readable, geometric, premium
- **Import**: `@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500&display=swap');`

### Typography Hierarchy

```
Display (H1)     → 4rem   Cormorant Garamond 400
Heading 1 (H2)   → 3rem   Cormorant Garamond 400
Heading 2 (H3)   → 2rem   Cormorant Garamond 400
Heading 3 (H4)   → 1.25rem Montserrat 600
Heading 4 (H5)   → 0.875rem Montserrat 600
Body             → 1rem   Inter 400
Body Small       → 0.875rem Inter 400
Caption          → 0.75rem Inter 400
```

### Typography Composition Rules

1. **Editorial Elegance**
   - Use Cormorant Garamond for all headings
   - Maintain generous line height (1.1-1.3)
   - Apply negative letter spacing for impact
   - Create visual hierarchy through size, not weight

2. **Clean Geometry**
   - Use Montserrat for UI elements and labels
   - Maintain consistent letter spacing (0.05-0.1em)
   - Use uppercase for emphasis
   - Keep geometric precision in alignment

3. **Readable Body**
   - Use Inter for all body text
   - Maintain 1.6 line height for readability
   - Limit line length to 65-75 characters
   - Use generous margins around text blocks

4. **Luxury Fashion Inspiration**
   - Pair serif headings with sans-serif body
   - Use generous whitespace around typography
   - Maintain strong visual hierarchy
   - Create elegant contrast between font families

## 📐 SPACING SYSTEM

### Spacing Scale

```
xs:   0.5rem   (8px)
sm:   1rem     (16px)
md:   1.5rem   (24px)
lg:   2rem     (32px)
xl:   3rem     (48px)
2xl:  4rem     (64px)
3xl:  6rem     (96px)
4xl:  8rem     (128px)
```

### Spacing Rules

#### Large Breathing Spaces
- **Section Padding**: 6rem (96px) vertical, 2rem (32px) horizontal
- **Card Padding**: 2rem (32px) minimum
- **Element Margins**: 1.5rem (24px) minimum between elements
- **Purpose**: Create premium, uncluttered appearance

#### Generous Section Padding
- **Top/Bottom**: 6rem (96px) for major sections
- **Left/Right**: 2rem (32px) for content padding
- **Mobile**: 3rem (48px) vertical, 1.5rem (24px) horizontal
- **Purpose**: Maintain visual balance and breathing room

#### Avoid Cramped Layouts
- **Minimum Gap**: 1.5rem (24px) between cards
- **Minimum Margin**: 1rem (16px) around text blocks
- **Minimum Padding**: 1.5rem (24px) inside containers
- **Purpose**: Ensure elegant, spacious composition

#### Maintain Visual Balance
- **Symmetry**: Equal spacing on opposite sides
- **Rhythm**: Consistent spacing patterns
- **Proportion**: Spacing relative to content size
- **Purpose**: Create harmonious, professional appearance

### Component Spacing

#### Cards
- **Padding**: 2rem (32px)
- **Gap Between Cards**: 2rem (32px)
- **Margin Bottom**: 2rem (32px)
- **Border Radius**: 0px (sharp edges)

#### Buttons
- **Padding**: 1rem (16px) vertical, 2rem (32px) horizontal
- **Margin**: 1rem (16px) between buttons
- **Border Radius**: 0px (sharp edges)
- **Min Height**: 48px (accessibility)

#### Form Elements
- **Padding**: 1rem (16px)
- **Margin Bottom**: 1.5rem (24px)
- **Border Radius**: 0px (sharp edges)
- **Min Height**: 44px (accessibility)

#### Text Blocks
- **Margin Bottom**: 1.5rem (24px)
- **Line Height**: 1.6
- **Max Width**: 65-75 characters
- **Padding**: 1rem (16px) around text

#### Lists
- **Item Spacing**: 1rem (16px)
- **Padding Left**: 1.5rem (24px)
- **Margin Bottom**: 1.5rem (24px)
- **Line Height**: 1.6

## 🎬 ANIMATION & MOTION

### Transition Timing

#### Smooth Transition
- **Duration**: 0.3s
- **Easing**: cubic-bezier(0.4, 0, 0.2, 1)
- **Usage**: Hover effects, color changes
- **Feel**: Responsive, elegant

#### Slow Transition
- **Duration**: 0.6s
- **Easing**: cubic-bezier(0.4, 0, 0.2, 1)
- **Usage**: Page transitions, major animations
- **Feel**: Cinematic, deliberate

#### Quick Transition
- **Duration**: 0.15s
- **Easing**: cubic-bezier(0.4, 0, 0.2, 1)
- **Usage**: Micro-interactions, feedback
- **Feel**: Snappy, responsive

### Animation Principles

1. **Cinematic Motion**
   - Smooth easing curves
   - Purposeful movement
   - Subtle, not flashy
   - Enhances user experience

2. **Minimal Visual Clutter**
   - No excessive animations
   - No gimmicky effects
   - No auto-playing videos
   - Focus on content

3. **Premium Feel**
   - Refined, elegant motion
   - Luxury fashion-inspired
   - Sophisticated transitions
   - Professional appearance
