## 🎨 DESIGN SYSTEM FEATURES

### Color Palette
- **Crushed Cacao** (#0F0807): Primary background
- **Mulberry Night** (#432430): Depth and secondary backgrounds
- **Indigo Tart** (#2A3548): Card surfaces and featured elements
- **Champagne Silk** (#CEB3AB): Primary highlights and accents
- **Glace Apricot** (#E8B4A8): Hover states and emphasis

### Typography
- **Headings**: Cormorant Garamond (400 weight, -0.02em letter-spacing)
- **Subheadings**: Montserrat (600 weight, uppercase)
- **Body**: Inter (400 weight, 1.6 line-height)

### Spacing System
- 8-level scale: xs, sm, md, lg, xl, 2xl, 3xl, 4xl
- Consistent editorial breathing room
- Responsive adjustments for mobile

### Shadow System
- **sm**: 0 5px 20px rgba(0, 0, 0, 0.2)
- **md**: 0 10px 40px rgba(0, 0, 0, 0.3)
- **lg**: 0 20px 60px rgba(206, 179, 171, 0.12)
- Cinematic layering for depth

### Animation System
- **Entrance**: Fade-in-up (0.8s ease-out)
- **Hover**: Lift (-4px) + shadow enhancement (0.3s)
- **Transitions**: Smooth (0.3s) for all interactive elements
- **Stagger**: 0.1s - 0.5s delays for sequential reveals
